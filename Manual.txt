How to add admin/vip commands on your own script?

Ģ Below of all the #include, Add #include <jadmin3>
Ģ You are now ready to use the functions available at the current include support+
Ģ To create an admin command, Simply use the CheckAdmin(playerid) function.

Ex:
if(strcmp(cmdtext, "/check", true) == 0)
{
if(CheckAdmin(playerid) >= 1)
{
SendClientMessage(playerid, -1, "/check is working.");
}
else
{
SendClientMessage(playerid, -1, "Not an admin level 1+");
}
return 1;
}

Ģ To create a vip command, do the same. Use CheckVip(playerid) function. [An example is already shown above, considered it as your base code]

Note; You may also set the admin/vip status of a player using SetPlayerAdmin / SetPlayerVip.
